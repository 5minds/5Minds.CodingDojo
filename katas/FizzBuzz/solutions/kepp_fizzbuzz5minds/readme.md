## Solution for the code kata 'FizzBuzz' with some special additions

> FizzBuzz is a group word game for children to teach them about division.

(Source: Wikipedia)

### Requirements

- Print numbers from 1 to 100
- Any number divisible by 3 is replaced by the word _Fizz_
- Any number divisible by 5 is replaced by the word _Buzz_
- Any number divisible by 3 and 5 is replaced by the word _FizzBuzz_

### Additional requirements

- Any number that contains 3 is replaced by the word _Fizz_
- Any number that contains 5 is replaced by the word _Buzz_
- Any number that contains both 3 and 5 is replaced by the word _FizzBuzz_

### How to run

- install [NodeJs](https://nodejs.org)
- navigate to solution directory
- run `npm install`
- to run the program run `npm run start`
- to run the test run `npm run test`
